3D file converter/library.

formats
  * import: .bos
  * import with Autodesk FBX SDK: + .fbx / .dae / .obj / .3ds
  * export: .bos
  * export with Autodesk FBX SDK: + .fbx

dependency
  * messagepack for bos
  * (optional) boost::python for pyd
  * (optional) Autodesk FBX SDK for FBX

compiler
  * Visual Studio 2013
